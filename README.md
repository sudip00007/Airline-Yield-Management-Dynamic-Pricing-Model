# ✈️ Airline Yield Management & Dynamic Pricing Simulation

## Overview

This project demonstrates an **Airline Yield Management and Dynamic Pricing Simulation** built using Python. The objective is to simulate how airlines can optimize ticket pricing based on factors such as demand, booking patterns, seat availability, and time remaining before departure.
The project applies data analysis, predictive modeling, and visualization techniques to illustrate revenue management concepts commonly used in the aviation industry.

## Features
* Dynamic airline ticket pricing simulation
* Demand forecasting and trend analysis
* Revenue optimization logic
* Data preprocessing and feature engineering
* Interactive visualizations
* Performance evaluation of pricing strategies
